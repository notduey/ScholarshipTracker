# scholarship-tracker
This is my first python project, this console program tracks the scholarships I add along with information including: name, link, amount, deadline, completion status, essay requirement, and additional notes. It supports adding, viewing, editing, and deleting scholarships, with all data stored in a CSV file for persistence. This project spanned a little over 3 weeks and there was very light AI assistance, mostly for sanity checks and structuring program, almost all of the logic was done by me albeit with minimal help from chatgpt.
